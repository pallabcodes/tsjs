# Ion Masterclass: Exhaustive Usage Examples

This directory contains a suite of "Google-Grade" examples that demonstrate the full architectural power of `@foculist/ion`.

## 📚 Chapters

| # | Example | Technical Focus |
| :--- | :--- | :--- |
| **01** | [Basic Validation](./01-basic.ts) | Flat schemas, rule chaining, and static inference. |
| **02** | [Deep Recursion](./02-recursion.ts) | Dot-notation path traversal and nested schema orchestration. |
| **03** | [Reactive Streaming](./03-streaming.ts) | Field-by-field feedback using the non-blocking event protocol. |
| **04** | [Advanced Logic](./04-advanced-logic.ts) | `compute`, `format`, `transform`, and conditional `omit` logic. |
| **05** | [Complex Reshaping](./05-recursive-mapping.ts) | The "Proto-to-UI" pattern for deep-to-flat data transformations. |

## 🚀 How to Run

These examples are written in TypeScript. You can run them directly using `tsx` or `ts-node`:

```bash
npx tsx examples/01-basic.ts
```
